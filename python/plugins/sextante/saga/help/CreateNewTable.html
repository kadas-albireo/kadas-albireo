Create New Table

O. Conrad (c) 2005
Creates a new empty table. Possible field types are:
- character string
- 1 byte integer
- 2 byte integer
- 4 byte integer
- 4 byte floating point
- 8 byte floating point
- 32 bit true color (RGB)
