OGR: Import Vector Data

O.Conrad (c) 2008
The "GDAL Vector Data Import" module imports vector data from various file/database formats using the "Geospatial Data Abstraction Library" (GDAL) by Frank Warmerdam. For more information have a look at the GDAL homepage:
http://www.gdal.org

Following vector data formats are currently supported:
