Export Image (bmp, jpg, pcx, png, tif)

O. Conrad (c) 2005
Saves a grid as image using display properties as used by the graphical user interface.

On the command line there are further parameters available to either use one of the default palettes, to use a Lookup Table for coloring or to interpret the grid as RGB coded.
