Lake Flood

Volker Wichmann (c) 2005-2010
This module can be used to flood a digital elevation model from seed points. Seed points have to be coded either with local water depth or absolute water level.

Parameters

Grid system
Grid system
Grid system
DEM
Input Grid
digital elevation model
Seeds
Input Grid
seed cells coded with local water depth or absolute water level, all other cells NoData
Absolute Water Levels
Boolean
check this in case seed cells are coded with absolute water level
Lake
Output Grid
extent of lake, coded with water depth
Surface
Output Grid
Flooded digital elevation model
