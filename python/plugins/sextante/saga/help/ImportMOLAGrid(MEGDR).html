Import MOLA Grid (MEGDR)

(c) 2003 by O.Conrad
Import Mars Orbit Laser Altimeter (MOLA) grids of the Mars Global Surveyor (MGS) Mission (Topographic maps, Mission Experiment Gridded Data Records - MEGDRs). Find more information and obtain free data from Mars Global Surveyor: MOLA (NASA)

Parameters

Grid
Output Data Object
File
File path
Grid Type
Choice
Available choices: 2 byte integer, 4 byte floating point
Orientation
Choice
Available choices: normal, down under
