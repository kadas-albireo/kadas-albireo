Export Point Cloud to Text File

V. Wichmann, LASERDATA GmbH (c) 2011
Exports a point cloud to a text file. Once the module is executed, a pop-up dialog allows to specify the fields to be exported and their decimal precision.

Parameters

Point Cloud
Input Point Cloud
The point cloud to export.
Text File
File path
The file to write the point cloud to.
Write Header
Boolean
Write column names.
Field Separator
Choice
Field Separator
Available choices: tabulator, space, comma
