Point Cloud Viewer

O. Conrad (c) 2009
This module is a 3D viewer for SAGA Point Clouds. Amongst others, the viewer supports attribute based coloring, a map view to control the area-of-interest, point size scaling based on distance, level of detail control and an anaglyph modus (to use with red/blue or red/green glasses).

Module development was supported by http://Laserdata GmbH, Austria
and http://alpS GmbH, Austria

Parameters

Points
Input Point Cloud
