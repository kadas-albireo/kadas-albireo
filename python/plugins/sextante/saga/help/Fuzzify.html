Fuzzify

Victor Olaya (c) 2004
Translates grid values into fuzzy set membership as preparation for fuzzy logic analysis.

Parameters

Grid system
Grid system
Grid system
Grid
Input Grid
Fuzzified Grid
Output Grid
Control Points
Node
A
Floating point
Values lower than A will be set to 0.
B
Floating point
Values between A and B increase from 0 to 1, values between B and C will be set to 1.
C
Floating point
Values between B and C will be set to 1, values between C and D decrease from 1 to 0.
D
Floating point
Values greater than D will be set to 0.
Membership Function Type
Choice
Available choices: linear, sigmoidal, j-shaped
Adjust to Grid
Boolean
Automatically adjust control points to grid's data range
