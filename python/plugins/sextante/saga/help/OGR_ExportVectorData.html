OGR: Export Vector Data

O.Conrad (c) 2008
The "GDAL Vector Data Export" module exports vector data to various file formats using the "Geospatial Data Abstraction Library" (GDAL) by Frank Warmerdam. For more information have a look at the GDAL homepage:
http://www.gdal.org

Following vector formats are currently supported:
