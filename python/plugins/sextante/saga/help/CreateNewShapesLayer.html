Create New Shapes Layer

O. Conrad (c) 2008
Creates a new empty shapes layer of given type, which might be either point, multipoint, line or polygon. Possible field types for the attributes table are:
- character string
- 1 byte integer
- 2 byte integer
- 4 byte integer
- 4 byte floating point
- 8 byte floating point
- 32 bit true color (RGB)
