Drop Point Cloud Attribute

Volker Wichmann (c) 2010, LASERDATA GmbH
The module can be used to drop an attribute from a point cloud. In case the output dataset is not set, the attribute will be dropped from the input dataset, i.e. the input dataset will be overwritten.

Parameters

Input
Input Point Cloud
Point Cloud to drop attribute from.
Attribute to drop
Table field
Attribute to drop.
Output
Output Point Cloud
Point Cloud with attribute dropped.
