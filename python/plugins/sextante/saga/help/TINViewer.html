TIN Viewer

O. Conrad (c) 2011
This module is a 3D viewer for SAGA TINs. Amongst others, the viewer supports attribute based coloring, a map view to control the area-of-interest, point size scaling based on distance, level of detail control and an anaglyph modus (to use with red/blue or red/green glasses).
